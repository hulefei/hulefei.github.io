学习观

什么是学习

记忆

  1. 容量有限
  2. 难以集齐

学习

无数情况和例子 压缩成 规律（知识）

先把书读厚，再把书读薄

  1. 问题
  2. 答案
  3. 知识描述
  4. 例子
  5. 区分例子 和 知识描述

方法

  1. 先尽可能的搜集更多例子帮助你体会问题和答案之间的关系
  2. 当真正学会时，例子被压缩成知识
  3. 学习是为了解决新问题
  4. 验证提炼出的知识，是否能描述 问题和答案之间的规律

学习步骤

  1. 明确问题和答案
  2. 用例子构建知识
  3. 验证知识有效性

学习误区

  1. 错误输入输出
  2. 错误学习方式
  3. 不用例子学习

学习最重要的两步

  1. 明确输入（问题）输出（答案）
  2. 用例子构建知识
    1. 知识构建分为两类，思考类和运动类，
    2. 思考类用推断问题如数学知识，运动类用直接反应如语言知识

分而治之

解决输入 到 输出 有两种方式：

  1. 增加知识范围
  2. 分而治之
    1. 将问题拆分成已有知识能解决的输入（问题）
    2. 适用所有思考类问题（编程无头绪，写作不出字，解题无思路，工作难应付）

思维导图

思维导图

  1. 列关键字
  2. 它是什么
  3. 它的目的
  4. 拆分知识

学习原则

  1. 明确任务输入输出
  2. 将信息压缩成知识
  3. 例子重塑大脑连接
  4. 二阶知识拆分知识
